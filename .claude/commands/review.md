description: Code review текущих изменений
---

Проведи code review выбранного кода или текущих изменений.

Проверь:
- Соответствие best practices Python/проекта
- Безопасность (SQL injection, XSS, etc.)
- Производительность и оптимизация
- Обработка ошибок
- Качество тестов
- Документация
- Названия переменных и функций
- Дублирование кода
- Соблюдение PEP 8

Предложи улучшения с рейтингом важности (critical/major/minor).

---
description: Генерация тестов для кода
---

Создай тесты для выбранного кода используя pytest.

Включи:
- Unit tests для всех функций/методов
- Edge cases и граничные условия
- Happy path и error cases
- Fixtures для test data
- Mocks для external dependencies
- Параметризованные тесты где уместно
- Понятные названия тестов (test_should_*)
- Докстринги для сложных тестов

Покрытие кода должно быть >80%.
